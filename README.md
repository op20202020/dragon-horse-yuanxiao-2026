# 龙马精神 · 2026元宵节庆祝网站

## 🎉 项目简介
这是一个为庆祝2026年元宵节特别设计的网站，主题为"龙马精神"。网站融合了中国传统文化元素和现代网页技术，营造出浓厚的节日氛围。

## 🏮 主题寓意
- **龙**：象征力量、智慧与吉祥
- **马**：代表速度、忠诚与成功  
- **龙马精神**：寓意昂扬向上、勇往直前的精神状态
- **元宵节**：象征团圆、和谐与新的开始

## 🎨 设计特色

### 视觉设计
1. **传统色彩**：使用中国红、金色等传统节日色彩
2. **动态元素**：灯笼、烟花、浮动祝福语等动画效果
3. **响应式布局**：适配各种设备屏幕
4. **交互动画**：悬停效果、加载动画、页面滚动动画

### 功能特性
1. **元宵倒计时**：实时显示距离2026年元宵节的时间
2. **互动灯谜**：传统灯谜游戏
3. **节日音乐**：背景音乐控制
4. **烟花特效**：点击页面触发烟花效果
5. **键盘快捷键**：便捷操作功能

## 🚀 使用方法

### 本地运行
1. 下载所有文件到同一目录
2. 用浏览器打开 `index.html`
3. 确保网络连接（需要加载外部资源）

### 功能操作
- **点击灯谜按钮**：显示/隐藏答案
- **点击音乐按钮**：播放/暂停节日音乐  
- **点击页面任意位置**：触发烟花效果
- **使用键盘快捷键**：
  - `M`：切换音乐
  - `F`：触发中央烟花
  - `1`：回到页面顶部

## 🔧 自定义修改

### 修改主题色
在 `style.css` 的 `:root` 部分修改CSS变量：
```css
:root {
    --primary-red: #d32f2f;      /* 主红色 */
    --gold-yellow: #ffd700;      /* 金色 */
    --dragon-green: #2e7d32;     /* 龙绿色 */
    --horse-brown: #795548;      /* 马棕色 */
}
```

### 修改倒计时日期
在 `script.js` 中修改元宵节日期：
```javascript
const yuanxiaoDate = new Date('2026-03-05T00:00:00');
```

### 添加更多灯谜
在 `index.html` 的互动区添加更多灯谜内容。
